# pse-che-lsu.github.io

This will be the webpage for the Process Systems Engineering group at LSU.

# 📁 Repository Structure

This repository is organized to keep content modular, readable, and easy to maintain.

## Root Directory

* **`index.html`**
  The main entry point of the site. This is what loads when visiting the GitHub Pages URL.

* **`README.md`**
  Provides an overview of the project, purpose, and key information for visitors and contributors.

* **`TODO`**
  A simple checklist or notes file for tracking pending tasks and future improvements.

---

## Folders

### 📦 `assets/`

Contains all static resources used by the website:

* Images
* Icons
* CSS stylesheets
* JavaScript files

Keeping these separate helps maintain a clean structure and improves scalability.

---

### 📄 `pages/`

Holds additional HTML pages beyond the main homepage:

* Subpages (e.g., about, research, contact)
* Organized content sections

This allows the site to expand without cluttering the root directory.

---

## 🧭 How It All Connects

* `index.html` serves as the homepage and links to files inside `pages/`
* `assets/` provides styling and media for all pages
* `pages/` contains structured content for navigation
* `README.md` documents everything for users and contributors

---

## 🚀 Purpose of This Structure

This layout is designed to:

* Keep files organized and easy to navigate
* Separate content from styling and assets
* Support future growth of the website

---

## ✏️ Notes

Feel free to expand the `pages/` directory or add more subfolders inside `assets/` (e.g., `css/`, `img/`, `js/`) as the project grows.
